# My Bedrock Project

Starter repo for AWS Bedrock + AgentCore hackathon project.

## Setup (local)

1. Create and activate venv:
   - `python3 -m venv .venv`
   - `source .venv/bin/activate` (macOS/Linux) or `.venv\Scripts\Activate.ps1` (Windows)

2. Install deps:
   - `pip install -r requirements.txt`
